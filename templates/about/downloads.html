Offline Server

For an offline capable version of the site for Windows, visit the download page. Unzip it and click on run.bat, which will start up the server and load the local version of the Khan Academy homepage in your browser. Next, go to the download_scripts directory and click on the playlist(s) you want to download (download_ALL.bat will get everything). When you're done, the videos you've downloaded will work when you click on them in the browser. Please note that this is a volunteer-developed version and is not officially supported.
